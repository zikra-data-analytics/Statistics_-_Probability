## 📚 Topics Covered

### 📐 Measures of Central Tendency
| Topic | Description |
|---|---|
| **Mean** | Average of data using `statistics.mean()` and `numpy.mean()` |
| **Median** | Middle value using `statistics.median()` and `numpy.median()` |
| **Mode** | Most frequent value using `statistics.mode()` and `statistics.multimode()` |

### 📏 Measures of Dispersion
| Topic | Description |
|---|---|
| **Range** | Max − Min using Python built-ins |
| **Variance** | Spread from mean using `numpy.var()` and `statistics.variance()` |
| **Standard Deviation** | `numpy.std()` and `statistics.stdev()` |

## 🧠 Key Concepts Learned

- **Mean** → Best for data with no extreme outliers
- **Median** → Best when data has outliers (e.g. house prices)
- **Mode** → Best for categorical data (e.g. most absent student)
- **Range** → Quick overview of total data spread
- **Variance** → How far each value is from the mean
- **Standard Deviation** → Average distance from the mean (most used in ML)
